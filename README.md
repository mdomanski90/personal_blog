# Self-hosted Personal Blog

Next.js blog z Keystatic CMS, Markdown i Docker.

## Stack technologiczny

- Next.js 16.1.6 (App Router + View Transitions API)
- Keystatic CMS
- React 18
- TypeScript 5
- Tailwind CSS 4
- Shuffle.js (animated filtering)
- PM2 (process manager)

Zarządzanie treścią

Posty blogowe zarządzane przez Keystatic CMS w trybie lokalnym.
Pliki markdown przechowywane w repozytorium.

100% FOSS
